Instructions

  1. Log in to the system using the prompt in the top righthand corner.
    • Should you not have a run identifier handy, use the Runs tab to either create a new run or get an old identifier
  2. The Results tab will give you a prettyprinted list of results for specified run.
  3. The Explore menu option will become available when you login.
  4. Once in the Explore tab you can:
    • Explore a topic in its particular corpus using the search interface
    • Try a Batch Judgement (outlined in the API section) request by clicking the associated link
